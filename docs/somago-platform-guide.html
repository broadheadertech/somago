What is Somago?

Full-stack e-commerce marketplace for Southeast Asia

Somago is a Shopee-style marketplace platform where buyers shop from thousands of products, sellers manage their stores and fulfill orders, and admins oversee platform health, moderation, and revenue. Built with Next.js, Convex, Clerk, and Tailwind CSS.

🛒 Buyer

Browse, search, buy, track orders, chat with sellers, leave reviews, earn rewards

🏬 Seller

List products, fulfill orders, track analytics, manage vouchers, API integrations

🛡️ Admin

Moderate content, manage users, review disputes, track revenue, fraud detection

⚙️

Tech Stack

Modern, scalable architecture

Frontend

Next.js 16, React, Tailwind CSS, shadcn/ui, Plus Jakarta Sans

Backend

Convex (real-time reactive database + server functions)

Authentication

Clerk (Phone OTP, Email, Google, Facebook)

Email

Resend (transactional emails: orders, approvals, disputes)

Payments

COD, Somago Balance, Stripe (cards), PayMongo (GCash/Maya)

Deployment

Vercel (web), Convex Cloud (backend), PWA support

Multi-Language & Multi-Currency

Currencies

Security

  • ✓
    Server-Side Price Validation
    All order amounts verified against database prices. Prevents client-side price tampering
  • ✓
    Role-Based Access Control
    Every Convex function validates user role. Seller functions check ownership. Admin functions check admin role
  • ✓
    XSS Prevention
    All user content escaped in emails (escapeHtml). Invoice HTML sanitized. URL validation on all user-submitted URLs
  • ✓
    Rate Limiting
    Batch operations capped (max 50). API key limits (max 10 per seller). Spin/check-in limited to once per day
  • ✓
    Duplicate Prevention
    Idempotent refunds, duplicate subscription checks, referral one-time use
  • ✓
    Fraud Detection
    Flag high-value orders, rapid purchases from new accounts, unusual patterns
