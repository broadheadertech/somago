1. Platform Overview

Somago is a full-stack e-commerce marketplace platform modeled after Shopee, designed for the Southeast Asian market. It connects buyers and sellers with comprehensive features for shopping, selling, and platform management.

User Roles

Role Description Access Level
Buyer Shop, browse, purchase products, leave reviews, chat with sellers Public pages + authenticated buyer features
Seller List products, manage orders, view analytics, run promotions, go live All buyer features + seller dashboard & tools
Admin Manage platform, moderate content, resolve disputes, view analytics Full platform access including admin dashboard

Security Measures

  • Server-side price validation on all orders
  • SellerId derived from product database, never trusted from client
  • Role-based access control (RBAC) on all mutations and pages
  • Rate limiting on chat (30/min), reviews (10/hr), disputes (3/hr)
  • XSS prevention: HTML escaping on emails, invoices, user content
  • URL validation: only http/https allowed for images and videos
  • Fraud detection: auto-flags suspicious orders
  • Duplicate refund prevention with idempotency checks
  • Voucher race condition prevention with atomic application
  • devSwitchRole gated behind development environment
